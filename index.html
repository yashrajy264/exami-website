For institutions — create, publish, manage

  1. Announce exam → fill details → attach policy docs
  2. Create the application form with blocks (fields, uploads, validations)
  3. Pick payment methods and fees
  4. Link exam centers (capacity, pin code rules)
  5. Publish. Push notifications to relevant candidates
