Roadmap to a Unified Exam Infrastructure

Accessible interactions, keyboard support, and motion that respects user preferences.

  1. lightbulb Early 2025 • Ideation
    Problem Discovery and Concept Validation

    Interviewed students, institutions, and policymakers; mapped fragmentation across application systems; identified reliability, accessibility, and security as the core pillars for Exami.

    Stakeholder interviews (6 states) Unified ID spec Compliance baseline
  2. science Mid 2025 • Planning & Research
    Architecture, Standards, and Partnerships

    Finalized system architecture for nationwide scale; researched integrations with Aadhaar/DigiLocker; began MoUs with pilot institutions and state boards.

    Data residency (India) Security controls 3 pilot institutions
  3. build Spring 2025 • MVP in Development
    Exami ID, Applications, Proctored Assessments

    Built core flows: unified Exami ID, application auto-fill, secure submissions, and AI-proctored exams. Added result delivery and certificate vault.

    Accessible UI 60fps target Observability
  4. rocket_launch 2026 & Ahead • Future Rollout
    State Pilots, National Programs, and Beta Iterations

    Expand pilots to multi-state programs, refine center operations, and roll out public beta with transparent SLAs and security posture.

    State pilots Center operations Security audits
